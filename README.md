## Vibemarks

A Nostr [NP-B0](https://github.com/nostr-protocol/nips/blob/master/B0.md) client, forked from [MKpinja](https://github.com/sepehr-safari/mkpinja).

In all honesty, you probably shouldn't be using or recommending this client. It's a quick and barely tested vibe coded fork of a vibe coded client. It's also probably not going to get much for updates. This is just a stopgap until some clients for bookmarking come along that do what I'm looking for.

That said, this is a fairly run of the mill public bookmarking service (think Hacker News) built on Nostr. It supports a few things I've yet to find a combination of in other clients:

* A focus on reading other public bookmarks
* NSEC bunker support
* A tag cloud + the ability to filter by tag
* The ability to use multiple relays
* Like/comment functionality
* URL query support (e.g. YouTube videos don't break)
